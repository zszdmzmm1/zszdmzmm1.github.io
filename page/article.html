国内执行npm指令

作者：

为什么慢

执行 npm 各种命令的时候，默认是去 npm 官方镜像源获取需要安装的具体软件信息 以下命令查看当前使用的镜像源 npm config get registry 默认源地址在国外，从国内访问的速度肯定比较慢

如何修改镜像源

淘宝维护着一个完整的 npm 镜像源 https://registry.npm.taobao.org/

a). 临时修改

npm install 软件名 --registry https://registry.npm.taobao.org/

b). 全局修改

npm config set registry https://registry.npm.taobao.org/

c). 使用第三方软件快速修改、切换 npm 镜像源

nrm NPM registry manager nrm 不仅可以快速切换镜像源，还可以测试自己网络访问不同源的速度

安装 nrm

npm install -g npm

列出当前可用的所有镜像源

nrm ls
    npm -----  https://registry.npmjs.org/
    cnpm ----  http://r.cnpmjs.org/
    taobao --  https://registry.npm.taobao.org/
    nj ------  https://registry.nodejitsu.com/
    rednpm -- http://registry.mirror.cqupt.edu.cn
    skimdb -- https://skimdb.npmjs.com/registr

使用淘宝镜像源
nrm use taobao

更多用法查看 nrm GitHub
